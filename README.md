# Free-Energy-Calculations
This repository hosts the Supporting Information (SI) files associated with the paper titled "Fine-Tuning Side Chain Substitutions: Impacts on the Lipophilicity–Solubility–Permeability Interplay in Macrocyclic Peptides". Specifically, it contains free-energy calculations for BE derivatives in POPC systems.
